# Form-Creation-Validation

A simple user registration form built with HTML, CSS, and JavaScript. This project implements basic client-side validation to ensure users provide valid information before registration is considered successful.

---

## Features

- **Username Validation**: Must be at least 3 characters.
- **Email Validation**: Must include "@" and "." characters.
- **Password Validation**: Must be at least 8 characters.
- **Real-time Feedback**: Displays clear messages for invalid inputs or confirms success.
